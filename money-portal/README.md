# 人生の宿題・診断ポータル(モック)

マネー講座「大人の義務教育」の内容をもとにした、受講者向け診断 & 代理店向け管理のシステムモックです。
`index.html` をブラウザで開くだけで動作します(外部ライブラリ・通信なし)。

## コンセプト

「無知はコスト」——受講者が自分の数字を計算し、**自分ごととして危機感と希望を持ち、セミナーへ足を運ぶ動機付け**を行う「人生の宿題解決プラットフォーム」。

## 画面構成(モックに含まれるもの)

| タブ | 内容 | 元ネタ |
|---|---|---|
| 🏠 診断 | 年齢・リタイア年齢・生活費などを入力 → 「人生の宿題(不足額)」を即時算出。年金ギャップ、インフレによる現金の目減り警告 | 概要資料 §1 |
| 📈 シミュレーション | 銀行0.2% vs 世界水準6%の積立比較、暴落時のドローダウン比較(「負けない運用」)、教育資金・保険見直しの応用例 | 概要資料 §2、講座 §3〜4 |
| 🔔 通知 | ライフイベント連動通知のプレビュー(教育・介護・円安・「夏休みの宿題」リマインド) | 概要資料 §3 |
| 🤝 代理店管理 | 紹介ファネル、診断完了率、フォロー優先度付きユーザー一覧、通知キャンペーン管理 | 概要資料 §4 |

## 元の概要に追加した独自アイデア

1. **宿題タイプ診断(ゲーミフィケーション)** — 準備ペースを夏休みの宿題に例えて「7月完了型/コツコツ型/8月31日型/新学期突入型」の4タイプに分類。診断結果に感情的なフックを持たせ、SNSシェアやセミナーでの話題化を狙う。
2. **「マイナスを抑える」体験クイズ** — 「-50%の後に+50%で元に戻る?」を対話的に体験させ、インデックス一辺倒との違い(講座§3)を1分で腹落ちさせる。
3. **必要積立額の逆算表示** — 不足額だけでなく「今から月いくら積み立てれば間に合うか」を貯金のみ/運用ありの2パターンで提示。行動(=セミナー参加)への距離を縮める。
4. **代理店向けフォロー優先度スコア** — 診断完了・宿題額・通知反応などの行動データから優先度を自動算出し、声かけのタイミングを逃さない。
5. **コンプライアンス配慮** — 免責事項・シミュレーションの前提明記・誇大表示を避けた文言をモック段階から組み込み(金融関連の集客システムでは必須)。
6. **将来拡張の想定** — LINE連携での通知配信、ねんきん定期便の読み取り入力、CRM連携、通知文言のA/Bテスト。

## 本実装時の想定構成(参考)

- フロント: 静的SPA(本モックはVanilla JS。実装時はReact等でも可)
- バックエンド: 診断結果の保存・通知スケジューラ・代理店別の紹介トラッキング(紹介コード付きURL)
- 通知: LINE公式アカウント / メール
- 注意: 投資助言・勧誘に該当し得る表現は金商法等の観点で要リーガルチェック
