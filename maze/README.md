# BASIC / 3D MAZE — Browser Prototype

元のBASICプログラムの「棒倒し法」を再実装した、静的HTMLだけで動く3D迷路です。

## 起動

`index.html` をブラウザで開いてください。Babylon.jsだけCDNから読み込みます。

ローカルサーバーで開く場合：

```bash
cd basic-3d-maze-browser
python3 -m http.server 8080
```

その後、`http://localhost:8080` を開きます。

## 操作

- `W` / `↑`：1マス前進
- `S` / `↓`：1マスあとずさり
- `A` / `←`：左へ90度回転
- `D` / `→`：右へ90度回転
- `Space` / `M`：マップ表示
- `R`：同じサイズ・Seedで再生成
- `Enter`：クリア後に次の迷路へ

移動は内部的にはマス単位、表示上は滑らかに補間します。カメラ揺れはありません。

## BASIC版との対応

- 初期サイズは7×7
- ゴールすると9、11、13、15へ拡大
- 15×15クリアで `CLEAR!`
- 迷路生成は、柱を偶数座標に置いて壁を1方向へ倒す棒倒し法
- 最初の柱列だけ4方向、以後は上方向を除く3方向
- 既存の壁へ重なる方向は選び直す

## リアルタイム調整

右側の `LOOK DEV PANEL` から、露出、Bloom、Fog、壁・床の質感、LED色、移動速度などを即時調整できます。

- `COPY LOOK JSON`：Unity移植用の設定をコピー
- `DOWNLOAD LOOK`：見栄え設定をJSONで保存
- `DOWNLOAD MAZE`：現在の迷路を完成済みグリッドとして保存

Unity側では乱数を再現するより、書き出した `grid` を読み込む方が確実です。

## Solid-block build（修正版）

- 2Dマップで `#` のセルは、3Dでもセル全体を占める閉じた直方体です。
- 薄い面を境界に貼る方式ではありません。
- 壁は結合せず、各セルごとに独立した固体ブロックとして生成します。
- 各ブロックには完全不透明の内部コアも入れてあり、向こう側が透けません。
- `app.js?v=solid-block-v3` として読み込むため、以前のJSキャッシュも回避します。
